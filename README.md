# Algorithm Study

알고리즘 문제 풀이 및 자료구조 구현 아카이브

## 폴더 구조

```
algorithm/
├── BOJ/                    # 백준
├── PGS/                    # 프로그래머스
├── LTC/                    # LeetCode
├── CFR/                    # Codeforces
├── implementations/        # 자료구조 및 알고리즘 직접 구현
└── README.md
```

## 커밋 메시지

```
플랫폼 문제식별
```

### 예시

```
[BOJ] 2178 미로탐색
[PGS] 완주하지못한선수
[LTC] 15 3Sum
[CFR] 580B KefaAndCompany
[impl] 레드블랙트리
```

### 파일 네이밍

| 플랫폼 | 파일명 형식 | 예시 |
|---|---|---|
| BOJ | `번호_한글명.py` | `2178_미로탐색.py` |
| PGS | `한글명.py` | `완주하지못한선수.py` |
| LTC | `번호_영문명.py` | `15_3Sum.py` |
| CFR | `라운드번호_영문명.py` | `580B_KefaAndCompany.py` |
| impl | `주제명.py` | `red_black_tree.py` |
